Music Downloader

The GUI was built using the Python library Custom Tkinter, which is a modern-looking version of the original Tkinter library. After entering the artist's name and clicking on the search button, a request is sent to the iTunes API website to retrieve information. The retrieved information is then analyzed and the relevant details are stored in an array which is displayed to the user. After selecting the desired songs by checking the checkboxes and clicking on the download button, musicD opens Google Chrome in the background and navigates to the website https://www.mp3juices.cc/2d0851 using the Selenium library for web scraping. The selected music files are then downloaded one by one before the browser is closed.
